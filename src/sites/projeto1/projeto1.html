O que é a propriedade Display em CSS:


CSS (Cascading Style Sheets) é usado para estilizar e organizar visualmente o conteúdo de uma página web.

Display Caracteristicas
Block Faz com que um elemento HTML seja renderizado como um bloco, ocupando toda a largura disponível do elemento pai e forçando uma quebra de linha antes e depois dele.
Inline Refere-se a elementos que ficam na mesma linha de outros, ocupando apenas a largura do seu conteúdo, sem forçar quebras de linha.
Inline-Block Permite que um elemento se comporte como um bloco, mas ainda assim se alinhe com outros elementos inline, como texto.
